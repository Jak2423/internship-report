\begin{abstract}
	Миний бие \@author \ "\@title" \ сэдэвтэй үйлдвэрийн дадлагын ажлыг “Зочил технологи” ХХК  компани дээр гүйцэтгэсэн. Энэхүү үйлдвэрийн дадлагын хүрээнд Javascript програмчиллын хэл дээр түлхүү ажилсан.

	Энэхүү дадлагын ажлын хүрээнд Bootstrap болон Javascript-н сан React дээр ажилсан билээ. Zochil платформын Админ вэб дээр react-hook-form ашиглан гараас хэрэглэгчийн оруулсан өгөгдлийг validation хийж backend-лүү дамжуулах, хэрэглэгчийн оруулсан өгөгдлийг backend-ээс авч жагсааж харуулах. Мөн Админ вэб-н responsive алдааг засах.

\end{abstract}
